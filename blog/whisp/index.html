Farhad Bigonahi ۳۰ تیر ۱۴۰۵ ۲ دقیقه مطالعه

Whisp (شلاق) یه اپ خیلی سبک و متن‌بازه که بعد از اجرا، بی‌سروصدا به‌صورت یه آیکون سفید داخل System Tray می‌شینه. هر وقت خواستی، روی آیکونش کلیک کن؛ یه شلاق آبی به موس وصل میشه و با تکون دادن موس، شلاق رو شبیه‌سازی می‌کنه. 💥

همزمان با صدای شلاق، یه پیام می‌فرسته و داخل پنجره‌ای که بازه تایپ میشه.

🚀 نصب

ساده‌ترین راه، نصب سراسری با npm‌ه:

bash
npm install -g whisp
bash
whisp

یا اگه دوست داری سورسش رو اجرا یا تغییر بدی:

bash
git clone https://github.com/FarhadBigonahi/Whisp.git
cd Whisp
npm install
npm start
